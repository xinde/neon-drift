# Neon Drift

霓虹漂移 — 一款赛博朋克风格的 2D 迷宫漂移游戏。

[在线试玩](https://xinde.github.io/neon-drift/) · [设计文档](DESIGN.md)

## 游戏玩法

- **手机**：倾斜设备控制光球漂移方向
- **PC**：WASD / 方向键控制
- **冲击波**：空格键（PC）/ 摇动设备（手机）触发，清除附近暗礁
- 收集全部晶体后抵达光门过关

## 开发

直接用浏览器打开 `index.html` 即可运行。

详细设计理念和维护文档见 [DESIGN.md](DESIGN.md)。

## 技术栈

- LittleJS v1.18.0（本地引擎）
- 原生 JavaScript，无构建工具
- GitHub Pages 部署
